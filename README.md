# ECommerce-in-ASP.NET
ECommerce functionalities done in ASP.NET.
A testing application made on ASP.NET with functionalities like :
1. DataList, 
2. GridView, 
3. Sessions,
4. Cookies, etc, 
implemented in various pages like 
Login, 
Register, 
Cart, 
View Products, 
Details, 
Filters, 
Cart,
and much more.
Styling is done using CSS and JavaScript.
